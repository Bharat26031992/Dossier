# Loading Screen

Simple loading screen demo implemented as a single `index.html` file.

## Project Overview

This repository contains a lightweight loading screen / splash screen demo. It is implemented with plain HTML, CSS, and (optionally) JavaScript inside `index.html`.

## Features

- Minimal, single-file demo (`index.html`).
- Easy to customize colors, animation, and text.

## Usage

1. Open `index.html` directly in a browser by double-clicking it.
2. Or serve it locally (recommended) if you want correct behaviour for assets or to test via HTTP:

   - Python 3:

     ```powershell
     python -m http.server 8000
     ```

   - Then open `http://localhost:8000/` in your browser.

## Development

- Edit `index.html` to change markup, styles, or animation.
- No build tools or dependencies required.
